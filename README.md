## 🎳 About Me 🎳

### 고태경 (2004.02.25)

안녕하세요.
코드를 작성하면서 동시에 그 결과로 만들어지는 데이터의 의미를 함께 고민하는 개발자 고태경입니다.

컴퓨터공학을 전공하며 C, Java, Python을 중심으로 프로그래밍의 기본기를 다져왔고,
기능 구현 과정에서 발생하는 로그, 상태 값, 실행 결과를 통해
시스템이 어떻게 동작하는지를 이해하는 데 관심을 가져왔습니다.

단순히 코드가 동작하는지에 그치지 않고,
어떤 데이터가 쌓이고, 그 데이터가 무엇을 말해주는지를 확인하며
문제의 원인을 찾고 구조를 개선하는 방식을 선호합니다.
필요한 경우 간단한 분석이나 정리를 통해 판단을 돕는 자료를 만들기도 합니다.

개발과 데이터 분석을 나누기보다는,
코드와 데이터가 서로 영향을 주고받는 과정으로 바라보며
읽기 쉽고, 설명 가능한 프로그램을 만드는 것을 목표로 하고 있습니다.

Hello,
I’m **Taegyeong Ko**, a developer who thinks about the **meaning of data generated by code** as much as the code itself.

With a background in computer science, I have built a solid foundation in C/C++, Java, and Python.
While implementing features, I pay close attention to logs, state changes, and execution results
to better understand how systems behave in practice.

Rather than stopping at “it works,”
I try to understand **what data is produced and what that data tells us**,
using it to identify issues and improve structure.
When needed, I organize or analyze results in simple ways to support clearer decisions.

I see development and data analysis as closely connected,
and I aim to write programs that are not only functional,
but also **readable, explainable, and informed by data**.
