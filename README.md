## 🎳 About Me 🎳

### 고태경 (2004.02.25)

안녕하세요.
코드를 작성하고 그 결과로 만들어지는 데이터를 분석하며 시스템을 이해하는 개발자 고태경입니다.

컴퓨터공학을 전공하며 C, Java, Python을 중심으로 프로그래밍의 기본기를 쌓았고,
개발 과정에서 발생하는 로그, 상태 값, 실행 결과 데이터를 기반으로
시스템의 동작을 해석하고 문제를 정리하는 데 익숙합니다.

기능이 정상적으로 동작하는지에 그치지 않고,
쌓이는 데이터를 직접 분석하며 원인을 확인하고,
필요한 경우 간단한 통계나 시각화, 머신러닝 기법을 적용해
패턴을 파악하거나 개선 방향을 고민해왔습니다.

개발과 데이터 분석, 머신러닝을 분리된 영역으로 보기보다는
코드 → 데이터 → 분석 → 개선으로 이어지는 흐름 속에서
읽기 쉽고, 근거를 설명할 수 있는 프로그램을 만드는 것에 관심이 많습니다.


Hello,  
I’m **Taegyeong Ko**, a developer who analyzes the data produced by code to better understand how systems behave.

With a background in computer science, I have built a solid foundation in **C, Java, and Python**,  
and I’m comfortable interpreting logs, state values, and execution results to reason about system behavior.

Rather than stopping at whether a feature simply works,  
I analyze the **data that accumulates during execution** to identify root causes.  
When appropriate, I apply basic data analysis techniques, visualization,  
and machine learning methods to explore patterns and consider possible improvements.

I view development, data analysis, and machine learning as a connected workflow,  
where code produces data and data informs better decisions.  
My focus is on writing programs that are **clear, explainable, and supported by evidence**.
